# Plans

Use this directory for dated planning packets.

Each plan folder uses this shape:

```text
plans/YYYY-MM/
  DD/
    change-id/
      plan.md
      implementation.md
      test.md
```

Add `survey.md` before `plan.md` only when the maintainer explicitly asks for a survey. Keep `test.md` as proof of working behavior, including commands run, expected results, actual results, and known gaps.
